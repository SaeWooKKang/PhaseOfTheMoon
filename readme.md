## About
문득 달을 관측할 수 있는 시간이 궁금해졌다. 공공데이터 포털에서 관련 api를 제공하는것을 찾아 제작하게 되었다. 

- 공공 데이터 포털의 API를 활용하여 월령과 달의 월출, 월중, 월몰 정보를 시각적으로 제공 
- 월령을 기반으로 달의 위상을 이모지로 표현
- 달의 위치를 시각적으로 표현
